
# Cybersecurity Interactive and learning Website: 

This is a fun website where users can interact with cool tools related to cybersecurity while learning valuable info about the topics with a short quiz at the end to test the users' knowledge. It has some cool tools that users can interact with and play with, as well as 4 basic interactive lessons of Cybersecurity

# Features of this Website:

- 4 basic lessons on Cybersecurity 
- Cool tools such as: Caesar Cipher, password strength checker, and Hashing Demo
- Firewall demo, phishing emails interactive and etc.
- A short 10-question quiz (added later) to test the concepts taught through lessons, Interactive, etc.
- A really cool UI (took a lot of time to code) :)

# How to use this Website:

Just click on this link below, scroll through the website and click on the lesson cards to learn more! 
Link: https://cybersecurity-interactive-and-info-website-otg6dqr2c.vercel.app
